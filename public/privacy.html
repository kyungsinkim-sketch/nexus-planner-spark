개인정보 처리방침

시행일: 2026년 4월 1일 | (주)파울러스

(주)파울러스(이하 "회사")는 Re-Be.io 서비스(이하 "서비스") 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다.

1. 수집하는 개인정보

필수 항목

  • 이메일 주소, 이름 (회원가입 및 로그인)
  • 비밀번호 (암호화하여 저장)

서비스 이용 과정에서 생성되는 정보

  • 프로젝트 데이터 (일정, 할 일, 채팅 메시지, 파일)
  • 이메일 연동 데이터 (Gmail OAuth — 사용자 동의 하에)
  • 통화 녹음 데이터 (사용자가 직접 시작한 경우에만)
  • 위치 정보 (출퇴근 기록 — 사용자 동의 하에)
  • 접속 로그, 기기 정보, IP 주소

2. 개인정보의 이용 목적

  • 서비스 제공 및 운영 (프로젝트 관리, 채팅, 일정 관리)
  • AI Agent(Brain AI) 기능 제공 (사용자 데이터 기반 분석 및 제안)
  • 사용자 인증 및 계정 관리
  • 서비스 개선 및 통계 분석
  • 고객 지원 및 공지사항 전달

3. 개인정보의 보관 및 파기

  • 회원 탈퇴 시 즉시 파기합니다.
  • 법령에 따른 보존 기간이 있는 경우 해당 기간 동안 보관 후 파기합니다.
  • 통화 녹음 데이터: 사용자 요청 시 즉시 삭제

4. 개인정보의 제3자 제공

회사는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우 예외로 합니다:

  • 사용자가 사전에 동의한 경우
  • 법률에 특별한 규정이 있는 경우

5. 개인정보의 위탁

  • Supabase (클라우드 데이터베이스 — 싱가포르 리전)
  • Vercel (웹 호스팅)
  • Google Cloud (음성 인식 STT — 비식별화 처리)
  • Anthropic (AI 분석 — 비식별화 처리)
  • LiveKit (실시간 통화 — 싱가포르 리전)

6. 이용자의 권리

  • 개인정보 열람, 수정, 삭제를 요청할 수 있습니다.
  • 회원 탈퇴를 통해 언제든지 개인정보 처리를 중단할 수 있습니다.
  • 데이터 이동권: 본인의 데이터를 내보내기(export)할 수 있습니다.

7. 개인정보 보호책임자

  • 회사명: (주)파울러스 (Paulus Co., Ltd.)
  • 대표: 김경신
  • 이메일: privacy@re-be.io

8. 데이터 보안

  • 모든 데이터는 전송 시 TLS 암호화를 적용합니다.
  • 비밀번호는 단방향 해시 알고리즘으로 암호화합니다.
  • 접근 권한을 최소한으로 제한합니다.
  • 정기적으로 보안 점검을 실시합니다.

Privacy Policy

Effective: April 1, 2026 | Paulus Co., Ltd.

Paulus Co., Ltd. ("Company") values the privacy of Re-Be.io ("Service") users and complies with applicable data protection laws.

1. Information We Collect

Required

  • Email address, name (for registration and login)
  • Password (stored encrypted)

Generated During Service Use

  • Project data (schedules, tasks, chat messages, files)
  • Email integration data (Gmail OAuth — with user consent)
  • Call recording data (only when initiated by user)
  • Location data (attendance — with user consent)
  • Access logs, device info, IP addresses

2. How We Use Your Information

  • Provide and operate the Service (project management, chat, calendar)
  • AI Agent (Brain AI) features (analysis and suggestions based on user data)
  • User authentication and account management
  • Service improvement and analytics
  • Customer support and notifications

3. Data Retention and Deletion

  • Data is deleted immediately upon account deletion.
  • Where required by law, data is retained for the statutory period then deleted.
  • Call recordings: deleted immediately upon user request.

4. Third-Party Disclosure

We do not share personal information with third parties without user consent, except:

  • When the user has given prior consent
  • When required by law

5. Service Providers

  • Supabase (Cloud database — Singapore region)
  • Vercel (Web hosting)
  • Google Cloud (Speech-to-text — de-identified)
  • Anthropic (AI analysis — de-identified)
  • LiveKit (Real-time calls — Singapore region)

6. Your Rights

  • Request access to, correction, or deletion of your personal information.
  • Delete your account at any time to stop data processing.
  • Data portability: export your data at any time.

7. Data Protection Officer

  • Company: Paulus Co., Ltd.
  • Representative: Kyungsin Kim
  • Email: privacy@re-be.io

8. Security

  • All data in transit is encrypted with TLS.
  • Passwords are hashed with one-way algorithms.
  • Access privileges are minimized.
  • Regular security audits are conducted.
